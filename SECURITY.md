# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in quantum-crypto,
please report it responsibly via email:

security@turboboost.tech

Please do not open a public GitHub issue for security concerns.

You will receive an acknowledgment within 48 hours and a
detailed response within 7 days outlining next steps.
